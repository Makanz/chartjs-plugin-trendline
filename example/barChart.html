Bar Chart with Trendline

Demonstrates linear trendline fitting on bar chart data

Configuration Used

This example shows a bar chart with a dotted trendline in pink color. The trendline is configured with:

  • colorMin: "rgba(255,105,180, .8)" - Pink semi-transparent color
  • lineStyle: "dotted" - Dotted line pattern
  • width: 2 - Line width of 2 pixels
