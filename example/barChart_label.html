Bar Chart with Trendline Label

Demonstrates trendline with custom label displaying percentage

Configuration Used

This example shows a bar chart with a dotted trendline that includes a custom label. The trendline is configured with:

  • colorMin: "rgba(255,105,180, .8)" - Pink semi-transparent color
  • lineStyle: "dotted" - Dotted line pattern
  • width: 2 - Line width of 2 pixels
  • label: Custom label with black text, percentage display, and 10px offset
