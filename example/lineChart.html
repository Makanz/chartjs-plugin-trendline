Line Chart with Labeled Trendlines

Multiple datasets with different trendline styles and custom labels

Configuration Features

This example demonstrates advanced trendline features on line chart data:

  • Multiple datasets: Each with unique trendline configurations
  • Custom labels: Asia dataset shows custom label positioning with offset
  • Legend integration: Custom legend text for trendlines
  • Line styles: Solid, dashed, and dotted trendlines
  • Color gradients: Asia trendline uses colorMin/colorMax for gradient effect
