📈 Exponential Trendline Example

What are exponential trendlines?
Exponential trendlines show data that increases or decreases at an exponential rate. They follow the equation y = a × e^(b×x) where data either grows rapidly (positive b) or decays (negative b).

Population Growth

Pattern: Exponential growth (10% per year)

Real-world example: Population growth in developing countries

Trendline: Solid red line shows the exponential growth pattern

Radioactive Decay

Pattern: Exponential decay (25% reduction per year)

Real-world example: Radioactive material half-life

Trendline: Dashed teal line shows the exponential decay pattern

🛠️ How to Use Exponential Trendlines

Step 1: Basic Configuration

Add trendlineExponential to your dataset configuration:

datasets: [{
    data: [...your data...],
    trendlineExponential: {
        colorMin: '#ff6b6b',      // Trendline color
        colorMax: '#ff6b6b',      // Same color for solid line
        width: 3,                 // Line thickness
        lineStyle: 'solid',       // 'solid', 'dashed', or 'dotted'
        label: {
            text: 'Growth Trend',
            display: true,
            displayValue: true,    // Show equation values
            color: '#ff6b6b',
            font: {
                size: 14,
                weight: 'bold'
            },
            offset: 15
        }
    }
}]

Step 2: Chart Type Considerations

Best practices:
  • Use scatter charts for precise point placement
  • Use line charts for time series data
  • Ensure your data has positive y-values for best results
  • Consider using logarithmic y-axis for very large value ranges

📊 Understanding the Output

Exponential Equation: y = a × e^(b×x)

  • a (coefficient): The initial value when x = 0
  • b (growth rate):
    • Positive b = exponential growth
    • Negative b = exponential decay
    • Larger |b| = faster change
  • e: Euler's number (≈ 2.718)

When to Use Exponential Trendlines

  • Population growth: Biological populations, user adoption
  • Financial compound interest: Investment growth, inflation
  • Technology adoption: Social media users, app downloads
  • Decay processes: Radioactive decay, cooling temperatures
  • Disease spread: Pandemic modeling (early stages)

// Solid line

lineStyle: 'dashed'

lineStyle: 'dotted'

Color Gradients

// Solid color
colorMin: '#ff6b6b',
colorMax: '#ff6b6b'

// Gradient effect
colorMin: '#ff6b6b',
colorMax: '#4ecdc4'

⚠️ Important Notes

  • Data requirements: Works best with positive y-values
  • Minimum points: Need at least 3 data points for fitting
  • Equation display: Set displayValue: true to show fitted parameters
  • Performance: Exponential fitting is computationally more intensive than linear
  • Accuracy: Works best when data truly follows exponential patterns
