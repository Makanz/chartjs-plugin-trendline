📊 Linear Trendline Example

What are linear trendlines?
Linear trendlines show relationships between variables that can be represented by a straight line. They follow the equation y = mx + b where m is the slope and b is the y-intercept.

🛠️ How to Use Linear Trendlines

Step 1: Basic Configuration

Add trendlineLinear to your dataset configuration:

datasets: [{
    data: [...your data...],
    trendlineLinear: {
        colorMin: '#ff6b6b',      // Trendline color
        colorMax: '#ff6b6b',      // Same color for solid line
        width: 3,                 // Line thickness
        lineStyle: 'solid',       // 'solid', 'dashed', or 'dotted'
        label: {
            text: 'Revenue Trend',
            display: true,
            displayValue: true,    // Show equation values
            color: '#ff6b6b',
            font: {
                size: 14,
                weight: 'bold'
            },
            offset: 15
        },
        legend: {
            display: true,
            text: 'Linear Trend (R² = 1.0)',
            color: '#ff6b6b'
        }
    }
}]

Step 2: Chart Type Considerations

Best practices:
  • Use scatter charts to show correlation between variables
  • Use line charts for time series data
  • Consider data distribution and outliers
  • Linear trendlines work best with roughly linear relationships

📈 Understanding the Output

Linear Equation: y = mx + b

  • m (slope): The rate of change - how much y changes for each unit change in x
    • Positive slope = positive correlation (y increases as x increases)
    • Negative slope = negative correlation (y decreases as x increases)
    • Steeper slope = stronger relationship
  • b (y-intercept): The value of y when x = 0
  • R² (coefficient of determination): How well the line fits the data (0 to 1)

When to Use Linear Trendlines

  • Sales analysis: Revenue vs marketing spend, price vs demand
  • Performance metrics: Experience vs productivity, training vs efficiency
  • Scientific research: Dose vs response, time vs distance
  • Economic indicators: GDP vs unemployment, inflation vs interest rates
  • Quality control: Process variables vs output quality

Correlation Strength Guidelines

  • R² > 0.8: Strong correlation - high predictive value
  • R² 0.5-0.8: Moderate correlation - useful for analysis
  • R² 0.2-0.5: Weak correlation - limited predictive value
  • R² < 0.2: Very weak correlation - consider other models

Legend Configuration

📊 Advanced Features

Projection Lines

Extend trendlines beyond your data range for forecasting:

trendlineLinear: {
    // ... other options
    projection: {
        enabled: true,
        range: [0, 100]  // Extend from x=0 to x=100
    }
}

Custom Formatting

label: {
    text: 'Custom Label',
    display: true,
    displayValue: true,
    format: function(value) {
        return 'y = ' + value.slope.toFixed(2) + 'x + ' + value.intercept.toFixed(2);
    }
}

⚠️ Important Notes

  • Data requirements: Need at least 2 data points (3+ recommended)
  • Outliers: Can significantly affect the trendline - consider data cleaning
  • Linearity assumption: Only use when the relationship is reasonably linear
  • Extrapolation caution: Predictions outside your data range may be unreliable
  • Correlation vs causation: A trendline doesn't prove causation
